YES — 색상만

데이터 흐름

1
호스트 → 에디터: content 전달 저장된 HTML을 에디터에 넣어줌 (수정 시)
2
사용자가 에디터에서 글 작성 타이핑, 이미지 삽입, 표 추가, 코드블록 등
3
에디터 → 호스트: onChange(html) 콜백 에디터 내용이 바뀔 때마다 HTML 문자열 전달
4
이미지/PDF 업로드 시 에디터가 onUploadFile(file) 호출 → 호스트가 서버에 업로드 → URL 반환 → 에디터에 삽입
5
저장 버튼 클릭 호스트가 form.content (HTML)를 sanitize 후 DB에 저장
6
게시물 열람 {@html content}로 렌더링 + hljs 코드 하이라이팅 + PDF 뷰어 마운트

다른 앱에 붙이는 법

5단계만 따라하면 됩니다

Step 1 — 설치

pnpm add @teriusu/hancomac-editor
# 또는
npm install @teriusu/hancomac-editor --legacy-peer-deps

Step 2 — CSS 변수 정의 (필수 8개)

:root {
  --background: #ffffff;
  --foreground: #2d3748;
  --primary: #4A7DAC;
  --muted: #f7fafc;
  --muted-foreground: #718096;
  --border: #e2e8f0;
  --popover: #ffffff;
  --destructive: #FF6B6B;
}

Step 3 — CSS import + Tailwind 소스

/* 글로벌 CSS */
@import "@teriusu/hancomac-editor/styles";

/* Tailwind v4 필수! 이거 없으면 메뉴가 투명 */
@source "../node_modules/@teriusu/hancomac-editor/dist";

/* Tailwind v3이면 대신: */
// tailwind.config.js content에 추가
// "./node_modules/@teriusu/hancomac-editor/dist/**/*.svelte"

Step 4 — 컴포넌트 사용

<script lang="ts">
  import { TipTapEditor } from '@teriusu/hancomac-editor';

  let content = $state('');

  // 업로드 함수 (선택)
  async function upload(file: File) {
    const fd = new FormData();
    fd.append('file', file);
    const res = await fetch('/api/upload', { method: 'POST', body: fd });
    return (await res.json()).url;
  }
</script>

<TipTapEditor
  {content}
  onChange={(html) => { content = html; }}
  onUploadFile={upload}
/>

Step 5 — 게시물에서 HTML 렌더링

<!-- 저장된 HTML 표시 -->
<div class="post-content">
  {@html savedContent}
</div>

<!-- 코드 하이라이팅 (선택) -->
import hljs from 'highlight.js';
import 'highlight.js/styles/atom-one-dark.css';
onMount(() => hljs.highlightAll());

요약

📦 에디터 = UI 완제품 (툴바, 메뉴, 스타일 내장)
🎨 호스트 = 색상 테마(CSS 변수) + 업로드 함수 제공
🔗 Tailwind 클래스는 호스트가 생성 → @source 필수
🔌 다른 앱에 붙이기: 5단계 (설치 → 변수 → import → 사용 → 렌더링)
